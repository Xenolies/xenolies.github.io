# 1.Akismet 反垃圾评论

直接注册可以用免费版本的，会给个 API 密钥，这个插件可以减少垃圾评论的骚扰。

一定要记住给的 API 密钥（好像没法找回）

# 5.WP Mail SMTP

一般博客网站会有博主回复的邮件提示，这个插件就可以实现这个功能。

可以申请个企业邮箱或者小号的邮箱来发邮件 2333

记得要设置下 SMTP 主机。

QQ 邮箱的话可以在 “设置” 中的 “账户‘进行设置。

阿里云企业邮箱的话，我一开始没找到服务器，然后没不知道 SMTP 密码。后来找帖子找出来了，就是下面这个。
