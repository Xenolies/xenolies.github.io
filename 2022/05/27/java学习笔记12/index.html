# 异常

异常是指程序运行当中出现的不期而至的情况。在程序运行期间，异常会影响正常程序的运行流程。

# 异常的分类

# 检查性异常

最具代表的时用户在错误或者问题的时候发生的，比如打开一个不存在的文件等

# 运行时异常

运行时异常是容易被忽略的异常，运行时可以忽略

# 错误 ERROR

错误不是异常，而是摆脱控制的问题。在代码中常常会发生。比如栈溢出等。

# 异常处理机制
