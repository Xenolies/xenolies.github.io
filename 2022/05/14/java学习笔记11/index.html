# 接口

JAVA 通过接口来实现多继承

# 定义接口

实现 (继承) 接口时，使用 implements 关键字来实现，而不是 extends 。

而接口继承接口时，使用 extends

# 内部类

内部类可以直接使用父类的私有 ( private ) 变量，而继承无法使用。

要在内部类方法中调用外部类中的值，使用 外部类名.this.变量名 来实现。
